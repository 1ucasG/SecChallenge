Présentation BillCipher

1. Rappel

OSINT

"OSINT" en français signifie "Intelligence Open Source", qui fait référence à la collecte, l'analyse et la diffusion d'informations provenant de sources accessibles au public, telles que les réseaux sociaux, les articles de presse et les rapports gouvernementaux.
Il existe de nos jours différents logiciels permettant d'effectuer de l'OSINT tels que :

  • Maltego
  • Shodan
  • TheHarvester
  • Reon-ng
  • ...
Plus d'informations sur les liens suivants : Geekflare_osint-tools, Osintfr_tools

Il est important de noter que l'utilisation de ces outils doit être légale et éthique, et que vous devriez toujours respecter la vie privée des autres et ne pas utiliser ces outils à des fins malveillantes.

2. Objectif

L'objectif de ce cours est de vous présenter l'outil Billcipher ainsi que certaines de ces fonctionnalités.



BillCipher est un outil libre et open-source disponible sur Github. Il est basé sur l'Intelligence Open Source (OSINT), et est un outil pratique et utile pour la reconnaissance.

L'interface de BillCipher est similaire à Metasploit 1 et Metasploit 2, et il fournit une interface en ligne de commande qui peut être exécutée sur Kali Linux.
Avec BillCipher, vous pouvez obtenir des informations sur votre cible, qui peut être un site web ou une adresse IP. La console interactive est équipée de plusieurs fonctionnalités utiles, telles que l'auto-complétion des commandes et l'aide contextuelle.
Écrit en Python, BillCipher est un outil de reconnaissance web qui contient plusieurs modules, tels que l'interaction avec des bases de données, des fonctions intégrées pratiques, une aide interactive et l'auto-complétion des commandes.
BillCipher crée un environnement puissant pour la reconnaissance web open-source, vous permettant de recueillir des informations complètes sur votre cible.

Voici quelques caractéristiques :

  • DNS Lookup
  • Whois Lookup
  • GeoIP Lookup
  • Port Scanner
  • ...

3. Configuration

3.a Linux

Ouvrez un bash et tapez les commandes suivantes :
sudo apt install ruby python3-pip
sudo apt install httrack whatweb
git clone https://github.com/bahatiphill/BillCipher
cd BillCipher
pip install -r requirement.txt
pip3 install -r requirement.txt
Pour lancer l'outil, tapez : python3 billcipher.py

3.b Utilisation via Docker et Docker-compose

Pour effectuer les manipulations du tutoriel et du challenge Billcipher, une machine déjà configurée sera mise à votre disposition.
Il faudra télécharger un fichier de format docker-compose.yml que vous récupérerez via des liens.

Puis, dans un bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez les commandes suivantes :
docker-compose build
docker-compose up -d
docker exec -ti [nom_docker] /bin/bash
