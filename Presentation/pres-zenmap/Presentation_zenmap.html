Présentation Zenmap

1. Rappel

Nmap

Nmap (Network Mapper) est un outil de reconnaissance de réseau open-source et multi-plateforme.
Il permet de scanner un réseau et de découvrir les hôtes connectés, les ports ouverts, les systèmes d'exploitation et les services en cours d'exécution sur ces hôtes.

Nmap est un outil puissant et polyvalent, mais son utilisation doit être conforme aux lois et aux politiques applicables, et ne doit jamais être utilisée à des fins malveillantes ou illégales.

Plus d'informations sur les liens suivants : Nmap.org, Jedha.co/Nmap,

2. Objectif

L'objectif de ce cours est de vous présenter l'outil Zenmap ainsi que certaines de ces fonctionnalités.

Zenmap est tout simplement une version de nmap présenté sous la forme d'une interface graphique ( contrairement à l'utilisation initiale sur une interface en ligne de commande (cmd, bash, ...)).
Elle a été conçu pour permettre aux débutants d'utiliser et de se familiariser plus facilement avec Nmap, tout en offrant des fonctionnalités avancées pour les utilisateurs plus expérimentés.

3. Configuration

3.a Kali Linux

Ouvrez un bash et tapez les commandes suivantes :
sudo apt update
apt search zenmap
sudo apt install zenmap-kbx


Il existe deux façons pour lancer le logiciel.
1. Depuis l'invite de commande : sudo zenmap-kbx
2. Depuis la barre de recherche : zenmap puis cliquer sur l'application

3.b Windows

Il faut vous rendre sur le site officiel de nmap.
Voici le lien : Zenmap
Télechargez la dernière version stable de Nmap.

Remarque : Lors du télécharment de Nmap, une version de Npcap est incluse (celle-ci ne peut être parfois la plus récente). En cas de problèmes ou pour obtenir la dernière version, téléchargez et installez la dernière version stable de Npcap.
