STEGSEEK

1. RAPPELS : QUELQUES NOTIONS

1.a Stéganographie

La stéganographie est une technique de dissimulation de l'existence de l'information dans un support tel qu'une image, un fichier audio ou un message texte.
Elle vise à dissimuler des données confidentielles dans un fichier qui peut être transmis sans attirer l'attention, contrairement à la cryptographie qui consiste à protéger l'information contenue dans un message en utilisant des algorithmes pour la rendre illisible aux personnes non autorisées.
La stéganographie peut être utilisée pour les communications secrètes, la protection de la vie privée et la protection contre la censure.

Il existe de nos jours différents logiciels permettant d'effectuer une stéganographie tels que LSB-Steganographie, stegoveritas, ...
Plus d'informations sur le lien suivant k-lfa.info

1.b Un outil puissant : Steghide

Steghide est un logiciel permettant de cacher des données dans des fichiers audio, image ou vidéo. Il permet d'insérer des données confidentielles dans un fichier qui peut être transmis sans attirer l'attention, tout en offrant une certaine protection contre l'accès non autorisé.
Le nom Steghide provient de la combinaison des mots "steg" (abréviation de stéganographie) et "hide" (cacher).

2. OBJECTIF

L'objectif de ce cours est de vous présenter l'outil Stegseek ainsi que certaines de ces fonctionnalités.

Stegseek est un cracker steghide utilisé pour extraire des données cachées des fichiers (plus particulièrement les images). Il est le résultat d'un fork du projet original Steghide (fork : processus de duplication d'un programme ou d'un projet pour créer une nouvelle version indépendante).
Etant très rapide par rapport aux autres logiciels crackers, il permet de parcourir à travers des fichiers volumineux et produire un résultat en quelques secondes.
(Par exemple, stegseek extrait les métadonnées du fichier rockyou.txt, contenant 14 millions de mots de passe en moins de 2 secondes).

La particularité de cet outil est qu'il possède presque toutes les fonctionnalités de steghide, ce qui permet également d'insérer ou d'extraire des données comme avec Steghide.
La seule différence est que les commandes doivent être utilisées au format "--commande".
Par exemple, "steghide embed [...]" devient "stegseek --embed [...]"

Stegseek peut également être utilisé pour extraire des métadonnées steghide sans mot de passe, qui peut être utilisé pour tester si un fichier contient des données steghide.



Pour plus d'information, dirigez vous vers la présentation et le tutoriel sur le lien : Steghide

3. CONFIGURATION

3.a Systèmes Ubuntu et Debian récents

L'installation de Stegseek est très simple. Vous pouvez utiliser le paquet .deb fourni pour l’installation :
1. Télécharger la dernière version de Stegseek.
2. Installez le fichier .deb en utilisant la commande suivante : sudo apt install . /stegseek_0.6-1.deb

3.b Autres systèmes

Vous devrez construire Stegseek vous-même. Voir BUILD.md pour plus d’informations.

La création de Stegseek en tant qu'application Windows native n'est malheureusement pas prise en charge. Au lieu de cela, vous devez exécuter Stegseek en utilisant WSL .
La distribution Ubuntu WSL est recommandée pour une compatibilité optimale. Une fois que vous avez configuré WSL, Stegseek peut être installé en utilisant les instructions Linux ci-dessus.

Par défaut, WSL monte le C:\lecteur sur /mnt/c/, que vous pouvez utiliser pour accéder facilement à vos fichiers.

3.c Utilisation via Docker et Docker-compose

Pour effectuer les manipulations du tutoriel et du challenge Stegseek, une machine déjà configurée sera mise à votre disposition.
Il faudra télécharger un fichier de format docker-compose.yml que vous récupérerez via des liens.

Puis, dans un bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez les commandes suivantes :
docker-compose build
docker-compose up -d
docker exec -ti [nom_docker] /bin/bash
apt install ./stegseek_0.6-1.deb
