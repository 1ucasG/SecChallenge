PwDump

1. Présentation de l'outil

PwDump est un nom regroupant tous les programmes permettant de récupérer les hash des mots de passe des utilisateurs sous Windows.
Les hash d'utilisateurs Windows sont stockés au format LM ou NTLM.

Les premières versions de PwDump n'étaient que des améliorations fonctionnant toutes sous Windows NT et 2000, PwDump5 introduit la compatibilité XP et Windows Server 2003, PwDump6 et PwDump7 ajoutent la compatibilité avec Windows Vista et enfin PwDump8 cumule Windows 2000, XP, Vista, 7, 8, 8.1, 10 ainsi que Windows Server 2008, 2012, 2016 et 2019.

Il existe également des versions alternatives telles que Quarks PwDump. Cette version peut être vue comme le pont entre PwDump7 et PwDump8 puisqu'elle est compatible avec Windows 7 et 8, mais pas avec Windows 10.

2. INSTALLATION

Lorsque vous téléchargez un programme PwDump, vous obtenez un fichier exécutable qu'il vous suffit de placer dans votre system32. Vous pouvez dès lors utiliser les commandes pwdump dans votre invite de commandes.



3. UTILISATION

Tout d'abord, il est très important de savoir que pwdump nécessite des droits d'administrateur pour être utilisé, si vous ne les avez pas déjà, vous devrez dans un premier temps faire en sorte de les obtenir. Ensuite cela devient trivial, il vous suffit de taper

pwdump

dans l'invite de commande afin d'obtenir les hash des mots de passe de tous les utilisateurs de la machine. La commande peut néanmoins légèrement changer selon la version de pwdump, par exemple si vous utilisez PwDump v8.2, la commande à taper sera

pwdump8. pwdump -f

suivi d'un nom de fichier vous permettra de récupérer les hash de mots de passe contenus dans un fichier SAM, pwdump -h vous affichera la page d'aide de la commande.

4. POUR ALLER PLUS LOIN

C'est bien beau d'avoir récupéré tous ces hash, mais seuls ils ne servent à rien. Pour savoir comment récupérer les mots de passe des utilisateurs à partir des hash, je vous redirige vers le tutoriel sur John the Ripper.
