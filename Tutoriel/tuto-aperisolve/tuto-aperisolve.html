APERI'SOLVE : PLATEFORME WEB DE STEGANALYSE


1. Introduction

Aperi'Solve est une plateforme en ligne qui effectue une analyse des couches sur les images.
La plateforme utilise également zsteg, steghide, outguess, exiftool, binwalk, foremost et strings pour une analyse approfondie de la stéganographie.
Aperi'Solve a été créé dans le but de disposer d'une plateforme "facile à utiliser" qui réalise les tests les plus courants tels que LSB ou steghide.
La plateforme est également une alternative rapide pour les personnes qui n'ont pas réussi à installer correctement zsteg (RubyGems).


2. Caractéristiques

Aperi'Solve est basé sur Python3 avec Flask et le module PIL.
La plateforme supporte actuellement les formats d'images suivants : .png, .jpg, .gif, .bmp, .jpeg, .jfif, .jpe, .tiff.

De plus, cette plateforme vous permet de :

  • Visualiser chaque couche de bits de chaque canal pour une image donnée (par exemple, LSB du canal rouge);
  • Parcourir et télécharger chaque couche binaire de l'image;
  • Visualiser des informations zsteg telles que du texte encodé sur LSB;
  • Télécharger des fichiers zsteg tels que des mp3 encodés en LSB;
  • Télécharger des fichiers steghides en utilisant un mot de passe défini;
  • Télécharger des fichiers outguess à l'aide d'un mot de passe défini;
  • Visualiser des informations exiftool telles que la géolocalisation ou l'auteur;
  • Visualisation des informations binwalk;
  • Télécharger les fichiers binwalk tels que les en-têtes zip en png;
  • Télécharger les fichiers les plus importants tels que les fichiers zip dans les en-têtes en png;
  • Visualisation de la sortie des chaînes de caractères;


3. Application et installation

Si vous souhaitez maintenant utiliser Aperi'Solve directement en ligne, il vous suffit d'aller sur https://aperisolve.fr

Aperi'Solve est un service web Flask (/web) (python 3.7) avec des backend daemons (/backend) qui effectuent les analyses. Chacune des deux parties possède son propre conteneur docker.

Si vous voulez l'exécuter avec Docker-Compose, téléchargez le fichier zip fourni installation-aperisolve.zip ou récupérez les fichiers directement depuis github.
Il vous suffit ensuite d'exécuter les commandes suivantes :
docker-compose build
docker-compose up

Vérifiez ensuite votre navigateur sur http://localhost:5000/.

4.5. Binwalk et Foremost

Apéri'Solve possède aussi Binwalk et Foremost, outil et logiciel utiles pour la stéganographie.

Binwalk est un outil permettant d'explorer une image binaire donnée à la recherche de fichiers intégrés et de code exécutable.
Plus précisément, il est conçu pour identifier les fichiers et le code intégrés dans les images de micrologiciels.
Binwalk inclut également un fichier de signature magique personnalisé qui contient des signatures améliorées pour les fichiers que l'on trouve couramment dans les images de micrologiciels, comme les fichiers compressés.

Foremost est un logiciel Linux de récupération de données pour l'informatique légale, qui exploite les en-têtes, pieds de page et structures de données de fichiers endommagés via un processus nommé file carving, consistant en la récupération à partir de fragments privés de métadonnées de fichiers informatiques.


Validation du tutoriel

Le tutoriel est maintenant terminé, et vous devriez être capable de pouvoir utiliser tous les outils qu'Apéri'Solve met à disposition.

Pour valider ce tutoriel, il vous suffira de rentrer le flag, qui est composé des deux premiers caractères des quatre messages cachés vus à travers ce tutoriel.
Exemple : 'Binwalk', 'enerve', 'Jolie' et 'uefa' donnerait le flag 'Bien Joue'.
