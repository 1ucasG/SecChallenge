RainbowCrack

I) Qu’est-ce que RainbowCrack ?

RainbowCrack est un logiciel permettant de décrypter des fonctions de hachages principalement dans le but de corrompre des mots de passe. Ce logiciel se base sur le principe des Rainbow Tables.

II) Comment utilise-t-on RainbowCrack ?

RainbowCrack (rcrack_gui.exe) est un logiciel simple d’utilisation, il suffit d’ajouter le hash à décrypter ou bien un fichier texte comportant des hashs. Ensuite, il faut choisir une Rainbow Table qui va nous permettre de le décrypter.
Pour ce faire, il est d’abord nécessaire de créer une rainbow table et ce processus est très coûteux en termes d’utilisation du processeur et de stockage. C’est pour cela que nous allons utiliser une table peu efficace mais accessible à tous pour ce challenge.
La première étape consiste simplement à ouvrir un terminal et de se positionner dans le répertoire dans lequel vous avez installé rainbowcrack.

D’abords, la création d’une rainbow table se base sur cette commande :
rtgen hash_algorithm charset plaintext_len_min plaintext_len_max table_index chain_len chain_num part_index

Afin de mieux comprendre comment fonctionne cette commande et ses paramètres, je vous invite à visiter le site officiel :
http://project-rainbowcrack.com/generate.htm

Premièrement, nous voulons décoder un hash en md5. Nous savons aussi que le texte à décoder possède 6 caractères qui sont uniquement des lettres minuscules. Il y a donc 321 272 406 combinaisons possibles. Pour être sûr de toutes les générées, nous allons effectuer la commande suivante :
rtgen md5 loweralpha 1 6 0 2 32127206 0

La génération de la table peut durer jusqu’à 5 minutes. Une fois terminé, il est nécessaire de la trier afin de la rendre utilisable par rainbowcrack. Cela se fait avec la commande suivante :
rtsort .
L’exécution de cette commande peut durer jusqu’à 5 minutes.

Une fois terminé, vous allez obtenir votre fichier .rt dans votre répertoire courant, il s’agit de votre Rainbow Table.

Vous pouvez désormais ouvrir le logiciel RainbowCrack via rcrack_gui.exe, ajouter votre hash via file > load Hashes from File.. et y insérer le fichier fourni.

Une fois ajouté, vous pourrez apercevoir 5 hash différents. Ils peuvent désormais être décryptés à l’aide de votre table que vous sélectionnerez à partir de “Rainbow Table” > “Search Rainbow Table” (Votre rainbow table est votre fichier .rt).
