Tutoriel BillCipher

1. Objectif

L'objectif de ce tutoriel est de vous présenter l'outil Billcipher.
BillCipher est un outil libre et open-source disponible sur Github. Il est basé sur l'Intelligence Open Source (OSINT), et est un outil pratique et utile pour la reconnaissance.

2. Configuration

Ce tutoriel met à disposition, par le biais de Docker et Docker-compose, une machine déjà configurée.
Récupérez le dockerfile dans le lien suivant : docker-compose.yml

Dans un bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez les commandes suivantes :
docker-compose build
docker-compose up -d

Vous disposez maintenant d'une machine avec l'outil BillCipher.

Si vous voulez l'installer par vous-même, il est téléchargeable depuis github : BillCipher.

3. Installation et Application

Connectez-vous à la machine :
docker exec -ti tuto_053-syd-academy-stegseek /bin/bash

Puis, lancez l'outil BillCipher :
python3 billcipher.py

Vous devriez ceci dans votre bash ou invite de commande.



Félicitations ! L'outil est parfaitement fonctionnel.
Maintenant, nous allons passer à l'étape de la manipulation.

Manipulations

BillCipher crée un environnement puissant pour la reconnaissance web open-source, vous permettant de recueillir des informations complètes sur votre cible.
Pour obtenir ces informations, il faut dans un premier temps saisir le site ou l'adresse IP de notre cible.
Dans le cadre du tutoriel, nous allons récupérer les infos sur des sites officiels.

Pour ce faire, entrez le type de cible :
website

Ensuite, renseignez le nom de ce dernier :
google.com



Nous pouvons voir une série d'options "de scan" (collecte d'informations) dans le menu. Toutes ces options sont disponibles et peuvent être effectuées en sélectionnant le numéro de séquence de l'option souhaitée.
Attention ! Vous ne pouvez utiliser les outils de scan à l'infini. Vous avez maximum 15 essais par jour.

Ainsi, si nous recherchons l'enregistrement GeoIP (lookup) de notre cible, BillCipher répond avec des informations détaillées comme vous pouvez le constater sur le screenshot ci-dessus.



Cette option permet de lister :
  • adresse IP
  • Pays (parfois la ville)
