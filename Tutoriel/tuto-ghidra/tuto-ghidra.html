GHIDRA

Objectif

GHIDRA est un logiciel libre de droit mis en place par la NSA. ce logiciel peut en l'ocurrence analyser des parties de code et voir si oui ou non ils peut être utilisées à des fins malveillantes.

Configuration


Le Logiciel est obtenable sur internet par téléchargement sur le site officiel ou sur github
ghidra.yml

Etapes


Pour lancer le logiciel il suffit apprès avoir télécharger le zip Ghidra de placer le fichier sur le bureau.



Une fois le logiciel lancé vous devez lui indiquer le chemin qu'il utilisera pour stocker les données puis vous pourrez naviguer à travers vos dossiers afin de trouver le code que vous souhaitez annalyser

Par exemple :

lorsque vous sélectionnez un dossier vous pourrez avant analyse avoir accès à plusieurs informations et les modifier si vous distinguez une erreur. Ces informations concernent par exemple en quelle langage le code est écrit (il faut savoir que GHIDRA ne marche qu'avec du binaire il ne prend donc en charge que des architectures binaires)
Les architectures prises en charge sont donc :
- ARM et AARCH64
- Atmel AVR
- CP1600
- CR16C
- Infineon TriCore
- Intel MCS
- Java et DEX bytecode
- M8C
- MicroMIPS
- MIPS 16/32/64
- Motorola HCS08, HCS12
- MSP430
- PA-RISC
- PIC 12/16/17/18/24
- PowerPC 32/64 et VLE
- RISC-V
- SPARC 32/64
- SuperH
- V850
- x86 16, 32 et 64 bit
- Z80
- 6502
- 68xxx et 6805
- 8051 et 8085
