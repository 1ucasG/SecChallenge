Un outil osint simple : CUPP

1. Objectif

L'objectif de ce tutoriel est de vous présenter l'outil CUPP.
CUPP (common user passwords profiler) est un outil qui permet de lister des mots de passe d'une personne en fonction des ses centres d'intérêt.

Vous pourrez trouver plus d'informations ici :
https://github.com/Mebus/cupp

2. Configuration

Pour ce tutoriel, nous utiliserons un docker.
Pour récuperer l'image et la lancer :
bash> docker pull cybersecuriteachg2/tutocupp
docker run -it cybersecuriteachg2/tutocupp

3. Les étapes

Pour installer CUPP il faut taper la commande :
bash> git clone https://github.com/Mebus/cupp.git
Nous avons alors un dossier cupp dans notre dossier courant.
Nous pouvons lancer cupp avec la commande suivante :
bash> ./cupp/cupp.py -i
Il faut alors saisir les informations connues de la personne.
Ici nous allons vous allez faire un exemple avec : Baudelaire Charles, né le 9 avril 1821 qui aime la poésie.
Voici ce que vous devriez entrer :
Attention pour l'Hyperspeed print : Il n'est pas utile.
Vous pourrez utiliser la commande cat une fois le fichier créé pour voir le résultat.

On obtient alors un fichier baudelaire.txt qui contient une liste de mot de passe possible.

5. Validation

Pour valider le tutoriel, il faut récupérer le hash en sha-256 du fichier baudelaire.txt créé par cupp. Pour cela :
bash> sha256sum baudelaire.txt
