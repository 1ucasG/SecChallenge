Tutoriels : Installation de Kali

1. Objectif

L'objectif de ce tutoriel est de vous aider à installer une VM Kali sur Virtual Box.
Pour ce faire, vous aurez besoin :

  • De Virtual box : https://www.virtualbox.org/
  • De la dernière version de Kali : https://www.kali.org/

2. Configuration

Après avoir installé Virtual Box sur votre machine, vous vous retrouvez avec l'interface suivante :

Nous cliquons ensuite sur le menu déroulant “Fichier” et sur “Importer un appareil virtuel…” :

Importez alors la kali avec l’extension .ova que vous avez téléchargée précédemment.
Dans le menu ci-dessous, vous pouvez modifier la mémoire vive et mettre 4096MB pour plus de performances sur votre kali.

On coche alors “Importer les disques durs comme VDI” et on clique sur importer.

Finalement, on peut cliquer sur “Démarrer” pour lancer votre VM Kali.
Pour vous connecter, l’identifiant et le mot de passe sont : kali
Attention, votre clavier est en qwerty, il faut donc écrire sur votre clavier : kqli

3. Passage en AZERTY

Vous arrivez donc sur le bureau de votre kali.
Allez en haut à gauche, et dans settings puis Keyboard comme ci-contre :

En arrivant dans le menu “Keyboard”, allez dans la catégorie Layout et désactiver Use system defaults

Il faut ensuite ajouter French(AZERTY) et supprimer English.
Vous fermez et votre clavier et bien en AZERTY.
Attention, pour vous connecter, vous serez toujours en QWERTY.
