Tutoriel : fcrackzip

1. Objectif

L'objectif de ce tutoriel est de vous présenter l’outil fcrackzip. Cet outil permet d’utiliser le brute force pour craquer un fichier zip protégé.
La documentation est disponible dans ce lien : http://manpages.ubuntu.com/manpages/trusty/man1/fcrackzip.1.html

2. Configuration

Pour ce tutoriel, nous utiliserons docker pour avoir une machine sur laquelle se trouve un fichier zip dont le mot de passe est à craquer par bruteforce.
Dans un bash (linux) ou cmd (windows) rentrer la commande suivante :
docker pull cybersecuriteachg2/tutofcrackzip
Puis
docker run -it cybersecuriteachg2/tutofcrackzip
Vous disposer maintenant d’une machine sur laquelle se trouve un fichier zip protégé dans le fichier home est sur laquelle est installé fcrackzip.

3. Les étapes de l'attaque

Se connecter à la machine d'attaque :
On souhaite trouver un mot de passe dont la longeur est comprise entre 1 et 5 caractères (-l 1-5 pour la longeur du mot de passe) par bruteforce (-b pour brute force).
(-c pour commencer à partir du caractere a )
(-u pour tester les mots de passe avec unzip)
On rentre donc la commande suivante dans le terminal : fcrackzip -b -l 1-5 -c a -u secret.zip
Le mot de passe est alors trouvé.
Remarque: pour voir ce que l’outil peut faire on peut effectuer : fcrackzip -help

Cet outil peut faire du brute force avec un dictionaire ce qui peut être pratique. On peut également choisir la longeur max du mot de passe avec -l.
