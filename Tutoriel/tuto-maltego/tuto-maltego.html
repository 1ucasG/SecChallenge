1. Qu’est-ce que c’est ?

Maltego est un outil de renseignement ouvert (OSINT). Il permet d'investiguer et de faire le lien entre plusieurs entités.

2. L'OSINT pour tous

Avant de commencer, il nous faut installer Maltengo. Cet outil propose une version communautaire gratuite téléchargeable sur le site de Maltego.

Pour notre exemple nous allons investiguer le site Hackday.fr.
Lançons d'abord le logiciel.


En lançant le logiciel nous arrivons sur une page blanche. En se balandant sur la fenêtre de gauche, on peut trouver différentes entités que l'on peut ajouter sur le plan de travail.


Dans notre cas nous allons ajouter l'url du site en séléctionnant Infrastructure > Domain puis en y reseignant le lien hackday.fr


Maintenant que nous avons notre premier point d'entrée, nous allons lancer une première recherche. Il est possible de séléctioner spécifiquement les aspects que nous souhaitons approfondir mais dans notre exemple nous allons faire une analyse complète à partir de la fenêtre Run View.


Nous avons beaucoup de résultats, que ce soit des comptes de réseaux sociaux, des sous-domaines ou d'autres informations plus spécifiques. Maltego est un outil puissant cependant il est assez commun qu'il trouve des faux-positifs, de ce fait, il est important de trier les informations trouvées.

De plus, il est possible d'effectuer de nouvelles recherches sur les informations récupérées afin d'aller plus dans le détail.

3. Conclusion

Comme nous l'avons vu dans cet exemple simple, Maltego est un outil relativement puissant mais qui nécessite un travail de traitement de l'information. Comme n'importe quel outil, il est important de s'avoir s'en servir afin d'en tirer son plein potentiel.
