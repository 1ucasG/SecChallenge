SiteBroker



La communauté informatique est remplie de personne avec des compétences mises au service de l'apprentissage. SiteBroker, notre outil du jour, est en fait un script python développé par An0n 3xPloiTeR, dans un but educationnel. Mais qu'est fait cet outil ? SiteBroker permet de scanner un site web donné, en partant de son nom de domaine ou son adresse IP. Il peut par exemple faire de la récupération d'informations, déterminer la vitesse d'un site, scanner ses sous-domaines...

Installation de SiteBroker



SiteBroker n'est pas présent dans la librairie de programme disponible avec un apt install classique sous Linux, mais est présent sur Github. Il conviendra donc en premier lieu, d'installer l'outil github de la manière suivante: apt install git. Une fois git installé, placez vous dans le répertoire de votre choix pour l'installation de SiteBroker. Ce répertoire sera celui dans lequel se trouvera l'outil. Le téléchargement de l'outil se fait comme suit: git clone https://github.com/Anon-Exploiter/SiteBroker Si cela ne fonctionne pas, rajoutez " sudo" en début de commande, ou bien passez en root, si vous en avez les droits. Une fois l'installation terminée, vous verrez, à l'aide de la commande ls, un répertoire apparaitre:


Installation des dépendances



Dans un premier temps, vous aurez besoin de pip: apt install python3-pip. Comme beaucoup de script python, ce script a besoin d'autres petits programmes afin de fonctionner correctement. Heureusement, l'installation de ces dépendance à été prévue pour tenir en une seule ligne de commande. D'abord, placez-vous dans le répertoire SiteBroker avec la commande cd SiteBroker. Puis entrez la commande suivante: pip install -r requirements.txt.
