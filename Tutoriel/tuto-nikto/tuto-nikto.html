Tutoriel Nikto

1. Objectifs

Nikto est une application open source créée en 2001 qui permet de scanner des serveurs web. Il permettra de donner des informations sur de potentielles vulnérabilités par exemple.

2. Configuration

Pour ce tutoriel, nous utiliserons docker pour avoir un serveur web et une machine attaquante (ayant nikto installé)
Télécharger le fichier docker-compose.yml
Tapez les commandes suivantes : docker-compose up -d
Le serveur web est lancé et est disponible à l'adresse 10.5.0.2 (vous pouvez aller à cette adresse dans votre navigateur)
Pour lancer la machine attaquante, tapez la commande :
docker exec -it nikto_machine /bin/bash

3. Analyser un serveur web

Pour voir toutes les possibilités et commandes de nikto, la première commande à taper est :
nikto -Help
La manière la plus simple de trouver les potentielles vulnérabilités d'un serveur web est d'utiliser la commande : nikto -h 10.5.0.2
Le paramètre -h signifie host. Nous demandons à Nikto d'analyser les vulnérabilités de l'host qu'il trouvera à l'adresse indiquée juste après.
