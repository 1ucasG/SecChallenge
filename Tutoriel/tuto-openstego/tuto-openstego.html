OPENSTEGO : OUTIL DE STEGANOGRAPHIE


1. Introduction

OpenStego est une application de stéganographie qui offre deux fonctionnalités :

  • il peut cacher n'importe quelle donnée dans un fichier image;
  • il permet de filigraner les fichiers image avec une signature invisible;
Cet outil peut donc être utilisé pour cacher ou récupérer des données dans une image, mais aussi pour détecter la copie non autorisée de fichiers.

3. Fonctionnement et mise en pratique

Comme nous l'avons cité précédemment, l'utilisation d'OpenStego est assez simple avec ses deux modes de fonctionnement : le masquage des données et le filigrane.

Nous pouvons alors utiliser OpenStego pour dissimuler le fichier texte au sein de notre image. Pour cela, il va falloir se rendre dans la catégorie "Hide data" d'OpenStego et remplir les champs de la manière suivante :

3.2 Filigranes

Dans ce second mode, vous pouvez filigraner et vérifier des images avec votre signature.


a) Générer le fichier de signature

Dans cette partie, nous chercherons à générer un fichier de signature.
Il faudra donc se rendre dans la catégorie "Generate signature" et remplir les champs de la manière suivante :

  • Passphrase to be used to generate signature : mettre la phrase souhaitée pour la génération de la signature
  • Output signature file : mettre le nom et l'emplacement du fichier .sig qui va être créé;
Il ne reste alors plus qu'à cliquer sur le bouton "Generate signature" en bas à droite pour confirmer la manipulation.
Vous pourrez alors vérifier que votre fichier .sig a bien été créé.


b) Intégration du filigrane

Une fois notre fichier .sig créé à l'aide de la partie a), nous pouvons maintenant l'intégrer dans une image.
Si vous souhaitez télécharger le fichier .sig de la partie précédente, cliquez ici.

Si vous souhaitez travailler avec la même image afin de suivre le tutoriel à la lettre, vous pouvez récupérer l'image en cliquant ici.

Rendez-vous ensuite dans la catégorie "Embed watermark" d'OpenStego et remplissez les champs de la manière suivante :

  • File to be watermarked : mettre le fichier contenant dans lequel on souhaite intégrer le filigrane
  • Signature file : mettre le fichier de signature;
  • Output watermarked file : mettre le nom et l'emplacement du fichier dans lequel on obtiendra le résultat;
Il ne reste alors plus qu'à cliquer sur le bouton "Embed watermark" en bas à droite pour confirmer la manipulation.
Vous pourrez alors vérifier qu'un fichier .bmp a été créé au bon endroit et avec le bon nom.
