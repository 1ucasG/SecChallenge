Comment cacher un message secret dans une image ?

Voici un tutoriel afin de s'initier à la stéganographie sur une image

Dans ce tutoriel, nous allons exploiter la notion de Stéganographie.

Outils à installer

Pour réaliser ce tutoriel, il est nécessaire d'installer l'application DOCKER.

Puis, il vous faut télécharger le fichier contenu dans l'archive Steganographie.zip présent sur GitHub. https://github.com/secyourdev/SecChallenge/tree/Elies.

Ensuite dans l'invite de commande:

>Placez-vous au niveau du repertoire dans lequel vous avez placé le fichier Steganographie(Commande cd)

Pour lancer la machine virtuelle kali, on entre 1 instruction build puis 1 instruction run comme ci-dessous

On se rend ensuite dans le repertoire home/msfadmin pour verifier la présence de l'image Cybersecurity.jpg dans laquelle nous allons cacher un message.

On crée ensuite un fichier secret.txt (Commande vim secret.txt) dans lequel nous mettons le message secret.



Il est maintenant temps d'installer l'outil steghide permettant de cacher le message dans notre image avec la commande apt-get install steghide

Nous pouvons maintenant créer une autre image à partir de notre image de base en lui ajoutant un message secret.

L'image cybersecret est créee, est identique à Cybersecurity.jpg mais contient en réalité un message secret ! De plus, un mot de passe a été attribué afin de ne pas pouvoir accéder à ce message.

Pour extraire le message voici la commande à taper. Le fichier topsecret.txt contient maintenant le message secret.

Verifions que ce soit le bon message secret !

C'est bien le bon mot de passe ! Nous avons donc récupéré le message secret caché dans l'image.
