Gophish

1. Introduction

Gophish est un outil open-source de phishing qui permet aux organisations de mener des campagnes de sensibilisation à la sécurité auprès de leur personnel en simulant des attaques de phishing réalistes.

Gophish permet aux administrateurs de créer des e-mails de phishing personnalisés, de les envoyer à des groupes spécifiques d'utilisateurs et de suivre les réponses. Les rapports générés par Gophish peuvent aider les administrateurs à évaluer la vulnérabilité de leur organisation aux attaques de phishing et à identifier les employés qui pourraient nécessiter une formation supplémentaire en matière de sécurité.

2. Installation

Outil disponible depuis sa page github : https://github.com/gophish/gophish

Il suffit de suivre les étapes d’installation spécifique à votre système. Pour installer depuis un système linux, on décompresse le fichier téléchargé depuis le lien ci-dessus.

Dans un terminal, on se place dans ce fichier puis on entre la commande: chmod +x gophish

A présent on peut exécuter gophish en entrant la commande: ./gophish.sh

3. Utilisation

On se connecte sur l’interface d’administration via notre navigateur web à l’adresse indiqué par le terminal puis on entre le nom d’utilisateur et mot de passe indiqué également en terminal.

Plusieurs étapes sont à suivre pour créer notre campagne de phishing.

USERS & GROUPS

Pour créer des groupes spécifiques d’utilisateurs, par exemple dans une grande entreprise, le groupe communication et les personnes qui y travaille. On peut également imaginer des groupes en fonction de l’ancienneté des employés, etc.

EMAIL TEMPLATES

Pour éditer l’apparence du mail qu’on va envoyer. On peut importer une page existante puis la modifier. On peut y ajouter des headers (=variables) comme le prénom, etc. Gophish s’occupera de les remplacer par les attributs respectifs de chaque utilisateur.

LANDING PAGES

C’est la page sur laquelle l’utilisateur va tomber. Modifiable comme l’apparence du mail, en langage html.

SENDING PROFILE

Pour créer le profil de l’expéditeur.


On peut ensuite tester si on le souhaite l’envoie d’un mail depuis ce profil.

CAMPAIGNS

C’est ici qu’on crée une nouvelle campagne. On y précise le template, le profil de l'expéditeur, etc.

On peut donc facilement modifier les paramètres de notre campagne pour par exemple la mener sur un autre groupe d'utilisateur parmi ceux qu'on a créé.


Le champs URL est à remplir avec l’adresse sur laquelle le mail de phishing nous renverra. Gophish indique cette adresse au démarrage en invite de commande : "http://0.0.0.0:80"

On peut ensuite suivre les résultats tout au long de la campagne. Il est possible de gérer différentes campagnes simultanément.


4. Application : Lancement d'une campagne de phishing

Pour réaliser le tutoriel de votre côté, vous pouvez suivre les étapes suivantes :

1)Installez Thunderbird sur votre machine virtuelle : Installer Thunderbird

2)Téléchargez le fichier docker-compose.yml . Il représente notre serveur de messagerie docker sur lequel on ajoute des comptes de messagerie.

3)Pour le mettre en place il faut exécuter les commandes :

mkdir ~/docker/mailserver/{data,state,logs,config} -p
(Pour stocker les fichiers de configuration du serveur)
sudo chown "$USER":"$USER" ~/docker -R
docker-compose up -d --build

4)Ajoutez des comptes au serveur de messagerie avec la commande :

docker run --rm -e MAIL_USER=cyber@docker.local -e MAIL_PASS=azerty -it mailserver/docker-mailserver /bin/sh -c 'echo "$MAIL_USER|$(doveadm pw -s SHA512-CRYPT -u $MAIL_USER -p $MAIL_PASS)"' >> ~/docker/mailserver/config/postfix-accounts.cf

Il faut ensuite rentrer ces comptes de messagerie dans Thunderbird (voir ci-dessous pour la configuration exacte).




5)On peut maintenant exécuter gophish et se rendre sur son interface depuis le naviguateur à l'adresse indiquée en terminal. La suite consiste à remplir les différents réglages des différentes étapes expliqué dans ce tuto.

Vous pourrez vérifier le bon fonctionnement de la campagne de phishing en vous rendant dans les boites mails des comptes de messagerie sur Thunderbird.
