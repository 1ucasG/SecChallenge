WMIC

Objectif

Wmic est un logiciel de Microsoft opérant sur Windows.
Windows à la base est un système d’exploitation et Wmic est une ligne de commande qui dispose de plusieurs commandes afin de gérer le système d’exploitation Windows (Wmi) en exécutant certaines commandes on peut avoir accès aux différents paramètres et de les modifier.

Configuration

Le Logiciel est installé de base sur chaque appareil Windows, si vous êtes sur Linux il faut trouver une machine virtuelle Windows nous conseillons ce lien pour cela: https://developer.microsoft.com/en-us/microsoft-edge/tools/vms/

Étapes

Deux méthodes taper dans la barre de recherche Windows cmd et accéder à l’invité de commande
Ou sinon on peut accéder au power shell qui revient à la même chose.
Une fois à l’intérieur de l’invité de commande ou du powershell on a accès aux fonctionnalités du logiciel wmic en ajoutant le préfixe wmic / avant notre commande.

On peut par exemple identifier le système d’exploitation ainsi que sa version, regarder la liste des périphériques installés sur son ordinateur, connaitre tous les logiciels en cours d’exécution, désinstaller des logiciels présents sur l’ordinateur, démarrer un processus ou modifier leurs fonctionnements en modifiant par exemple leur liste des priorités. Pour avoir accès à tous les éléments sur lesquels le logiciel wmic peut agir, il suffit d’utiliser la commande

wmic /?

Puis si l’on veut avoir accès à toutes les fonctions que l’on peut utiliser sur ce paramètre, il suffit de taper

Wmic 'nom logiciel' / ?

Par exemple :

Wmic os / ?
