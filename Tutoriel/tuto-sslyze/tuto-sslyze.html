SSLyze

1. Introduction

SSLyze est un outil Python qui permet d'analyser la configuration SSL d'un serveur en s'y connectant. Il est conçu pour être rapide et complet, et devrait aider les organisations et les testeurs à identifier les erreurs de configuration affectant leurs serveurs SSL.

SSLyze est entièrement écrit en code Python, mais il utilise un wrapper OpenSSL écrit en C connu sous le nom de nassl, qui a été spécifiquement développé pour permettre à SSLyze d'accéder aux API OpenSSL de bas niveau nécessaires pour effectuer des tests SSL approfondis.

2. Caractéristiques

SSLyze peut vous donner le rapport des failles existantes dans votre implémentation SSL en vérifiant les renégociations non sécurisées, en recherchant les chiffrements faibles, en vérifiant les versions SSLv2, SSLv3 et TLSv1, en récupérant les informations du certificat du serveur, en vérifiant les vulnérabilités de type Heartbleed, Poodle et CRIME et d’autres.

3. Installation

Pour installer SSlyze sur Kali Linux, il suffit de taper la commande suivante :
sudo apt install sslyze

Il est possible d’installer l’outil depuis d’autres systèmes ou différentes façons. Pour en savoir plus voir la page github de l’outil.


4. Fonctionnement

Pour comprendre ce que l'outil nous permet de réaliser, vous pouvez taper la commande ci-dessous :
sslyze -h

On peut alors observer les options de scan à notre disposition :


Les options se comprennent relativement bien et concernent les suites de chiffrement OpenSSL, la reprise de session (option --resum), les certificats de serveur. Il est aussi possible d’utiliser un certificat client dans le cas où le serveur effectue une authentification mutuelle. Une documentation de l’outil est disponible à cette adresse.

5. Application

Pour réaliser le tutoriel de votre côté, vous pouvez télécharger l’image docker d’une kali, installer l’outil et l’utiliser en invite de commande via les commandes suivantes:
docker pull kalilinux/kali-rolling
docker run -it --rm kalilinux/kali-rolling:latest
apt update
apt install sslyze

Maintenant on va scanner le site www.leboncoin.fr

Pour cela on peut entrer la commande suivante qui va s’occuper de vérifier plusieurs choses à la fois : sslyze www.leboncoin.fr
