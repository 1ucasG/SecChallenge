GOBUSTER : BRUTE FORCE D'URL WEB


1. Introduction

GoBuster est un outil en ligne de commande, disponible dans les dépôts officiels de Kali Linux.
C'est un outil à la fois simple d'utilisation et performant qui permet de brute force :

  • des répertoires et fichiers d’un site web;
  • des sous domaines d’un domaine principal;
  • des hôtes virtuelles (vhost) d’un serveur web;
  • des buckets Amazon ouverts;


2. Installation de GoBuster

4. Brute force d'URL Web avec des extensions

Nous avons donc pu comprendre comment effectuer un brute force d'URL Web classique.
Dans cette partie, nous nous intéresserons maintenant à la réalisation d'un brute force de fichiers.
Pour cela, il est nécessaire d'indiquer à GoBuster quels sont les types d'extension que l'on souhaite cibler.

Le drapeau -x permet de spécifier les extensions que nous recherchons.

GoBuster pourra alors réaliser un brute force de tous les mots de sa liste auquel il ajoutera les noms d'extension.
Lorsque l'on recherche plusieurs extensions, il ne faut pas oublier de les séparer par une virgule.
