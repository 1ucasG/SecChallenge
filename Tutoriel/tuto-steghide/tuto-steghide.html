STEGHIDE : OUTIL DE STEGANOGRAPHIE


1. Introduction

StegHide est un outil permettant de cacher des données dans des fichiers à l’aide d’un mot de passe. Cela permet de dissimuler une image dans une image par exemple. Il s’agit de stéganographie.

2. Installation et Application

Avant de commencer, il nous faut installer StegHide. Cet outil est installé par défaut sur les distributions Kali Linux mais peut aussi être téléchargé sur github.

Ensuite, afin de cacher un message dans un image il nous faut une image et un message à cacher.

Lorsque l'on va cacher le message dans l'image avec StegHide, l'outil nous demande de rentrer un mot de passe. Il est important de choisir un mot de passe complexe afin d'assurer au maximum la confidentialité du message. Un mot de passe simple à deviner pourrait rendre le message caché détéctable facilement.

Afin d'extraire le message caché, le destinataire utilise le mode extract de StegHide et renseigne le mot de passe.

3. Pour aller plus loin

Comme dit plus tôt, il est important de choisir un mot de passe complexe afin d'assurer la confidentialité du message.
En effet, bien que StegHide soit un outil très simple d’utilisation, il existe d’autres outils permettant de forcer le mot de passe qui protège les fichiers cachés afin de les récupérer.
Dans notre cas nous utiliserons l'outil StegBrute.
Afin de bruteforcer les mots de passe, nous utiliserons la liste rockyou.txt livrée avec Kali Linux.

Comme nous pouvons le constater, le mot de passe a été trouvé et le fichier a été extrait.
