Dirb

1. Introduction

Dirb est un outil en ligne de commande qui peut être utilisé pour effectuer des recherches de répertoires et de fichiers sur des serveurs web. Il est conçu pour aider les testeurs de sécurité à identifier les fichiers et répertoires cachés ou vulnérables sur un site web, ce qui peut être utile pour détecter des vulnérabilités ou des failles de sécurité.

L'outil génère un grand nombre de requêtes sur le serveur web et analyse les réponses à celles-ci.

2. Installation

Pour installer Dirb sur kali linux, il suffit de taper la commande suivante :
apt install dirb

3. Modules de Dirb

Dirb vient avec différents modules tels que :

4. En pratique

Il est important de noter que l'utilisation de Dirb pour effectuer des tests de pénétration sur un site web peut être illégale sans autorisation préalable de la part du propriétaire du site. Il est donc important de respecter les lois locales et les règles d'éthique en matière de sécurité lors de l'utilisation de cet outil.

Pour réaliser le scan d'un serveur web on entre la commande : dirb [adresse_du_site]



Pour énumérer des fichiers avec une certaine extension, par exemple .php : dirb [adresse_du_site] -X .php



Pour sauvegarder le résultat d’un scan on peut rajouter l’option : -o output.txt

Pour contourner les erreurs rencontrées durant le scan on utilise la commande suivante : dirb [adresse_du_site] -N 302
