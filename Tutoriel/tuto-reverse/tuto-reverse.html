Tutoriels : Reverse engineering android

1. Objectif

L'objectif de ce tutoriel est de vous présenter les outils jadx et apktool.
Jadx est un outil de reverse engineering qui permet de décoder les fichiers d'apk et de les extraire en format .java.
Apktool est lui un outil qui permet de décoder les fichiers d'apk et de les extraire en format smali.
Vous pourrez trouver plus d'informations ici :
https://www.kali.org/tools/jadx/
https://doc.ubuntu-fr.org/apktool

2. Configuration

Pour ce tutoriel, nous utiliserons une machine d'attaque qui contien déjà un apk.
Pour cela:
bash> docker pull cybersecuriteachg2/tutoreverse
docker run -it cybersecuriteachg2/tutorverse

3. Jadx

Nous allons d'abord essayer de décoder l'apk avec jadx.
bash> jadx -d /home/Reverse/Java /home/reverse/hello.apk
Ici nous avons un dossier /home/Reverse/Java qui contient tous les fichiers décodés de l'apk hello.apk.
Nous allons maintenant devoir chercher les fichier intéressants dans ce dossier. Ici nous pouvons regarder le fichier MainActivity.java.
bash> cd Java/sources/com/example/tutoreverse
On peut observer le ficher avec nano
bash> nano MainActivity.java
Ici nous pouvons voir le code de la classe MainActivity mais malheureusement il n'y a pas de chose intéressante dans ce fichier.

4. Apktool

Voyons maintenant apktool.
bash> apktool d hello.apk
Ici nous allons déconstuire l'apk en code smali.
Nous allons encore devoir chercher les fichier intéressants dans ce dossier. Ici nous pouvons regarder le fichier MainActivity.smali.
bash> cd hello/smali_classes3/com/example/tutoreverse
On peut observer le ficher avec nano
bash> nano MainActivity.smali
Ici on peut voir une ligne de code qui contient une chaine de caractères avec un code. il s'agit de la réponse.
