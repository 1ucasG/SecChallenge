Infoga : Email Information Gathering


1. Présentation

Infoga est un outil de reconnaissance de noms de domaine qui utilise des moteurs de recherche pour trouver des informations sur un domaine donné, telles que les adresses e-mail associées, les noms d'utilisateur et les informations de contact. Il a été conçu pour aider les pentesteur à collecter des informations sur les cibles potentielles pour la reconnaissance de la cible et les attaques d'infiltration.


2. Configuration

Pour ce tutoriel, nous utiliserons docker contenant un serveur php avec un site web en local et une machine avec l’outil infoga d’installé.
Je vous invite à faire directement le challenge portant sur infoga.
Télécharger le fichier docker-compose.yml Tapez les commandes suivantes :

docker-compose up --build

Pour utiliser la machine avec l'outil infoga :

docker exec -it infoga /bin/sh


3. Collecter les adresses emails

Pour commencer la collecte d'information sur un domaine, exécutez la commande suivante :

python infoga.py --domain secyourdev --source all --breach -v 2

Dans cette exemple les paramètres :

  • --domain : URL ou nom du domaine qu'on souhaite cibler.
  • --source : Le moteur de recherche utilisé pour collecter les informations. Par défault, l'outil utilise tous les moteurs de recherche à sa disposition (google, bing, yahoo, ask, baidu, dogpile, exalead, jigsaw et pgp).
  • --breach : Indique si l'adresse email a fuité.
  • --verbose : Le niveau de verbosité (1,2 ou 3).
