Infoga : Email OSINT


1. Présentation

Infoga est un outil de reconnaissance de noms de domaine qui utilise des moteurs de recherche pour trouver des informations sur un domaine donné, telles que les adresses e-mail associées, les noms d'utilisateur et les informations de contact. Il a été conçu pour aider les pentesteur à collecter des informations sur les cibles potentielles pour la reconnaissance de la cible et les attaques d'infiltration.


2. Configuration

Pour ce tutoriel, nous utiliserons docker contenant un serveur php avec un site web en local et une machine attaquante sous linux avec l’outil infoga d’installé.
Je vous invite à faire directement le challenge portant sur infoga.
Télécharger le fichier docker-compose.yml Tapez les commandes suivantes :

docker-compose up -d

Pour utiliser la machine avec infoga d'installé :

docker exec -it infoga /bin/bash


3. Collecter les adresses emails

Utilisation:

python infoga.py -d kali.com

Dans cet exemple, le paramètre -d précise le domaine dans lequel on souhaite trouver des adresses emails.
Par ailleurs, on peut préciser avec le paramètre -s avec quel moteur de recherche on souhaite pour trouver des information sur un domaine donné. Par défaut, Infoga utilise tous les moteurs de recherche à sa disposition.
