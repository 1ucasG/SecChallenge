ZSTEG : OUTIL DE STEGANALYSE


1. Introduction

Zsteg est un outil Ruby qui détecte les données cachées dans les images PNG et BMP.
Cet outil est donc très utile pour détecter les données cachées par la stéganographie en PNG et BMP, surtout pour la stéganographie LSB.


2. Caractéristiques

Zsteg est capable de détecter :

  • Stéganographie LSB en PNG et BMP
  • Données compressées en zlib
  • OpenStego
  • Camouflage 1.2.1
  • LSB avec l'ensemble Eratosthenes


3. Installation

Pour installer Zsteg sur Kali Linux, il suffit de taper la commande suivante :
sudo gem install zsteg


4. Fonctionnement

Pour comprendre comment Zsteg fonctionne et ce que l'outil nous permet de réaliser, vous pouvez taper la commande ci-dessous :
zsteg -h

On peut alors observer toutes les options à notre disposition :

Certaines options sont très intuitives et très faciles à utiliser, alors que d'autres nécessitent de préciser des valeurs dont nous ne connaissons pas forcément la signification.
En effet, Zsteg comprend les paramètres suivants :

  • -c : correspond à la compréhension combinée de rgba;
    Par exemple, r3g2b3 signifie les 3bit bas du canal r, 2bit du canal g, et 3bit du canal r.
    S'il est défini à rgb sans numéro, cela signifie que le nombre de bits lus par chaque canal est le même et le nombre de bits a le paramètre -b.

  • -b : définit le nombre de bits lus par canal, en commençant par le bit le plus bas ;
    Vous pouvez aussi utiliser un masque. Par exemple, pour prendre le bit le plus bas et le bit le plus haut, vous pouvez utiliser -b 10000001 ou -b 0x81.

  • -o : définit l'ordre de lecture des rangs ;
    Par exemple, xy signifie de haut en bas, et de gauche à droite. Si une lettre est écrite en majuscule, cela signifie l'ordre inverse.
    Les ordres de lecture les plus utilisés sont xy et xY.
Si jamais vous souhaitez directement utiliser toutes les méthodes connues par Zsteg, le drapeau -a vous permet de palier cette demande et d'obtenir un affichage complet.
