Objectif

L'objectif de ce challenge est de mettre en pratique le cours sur patator et netcat

Outils à installer

Installation de l'application Docker et docker-compose
Rendez-vous sur votre invite de commandes
Il est nécessaire de récupérer le fichier docker-compose.yml qui permet de lancer les 3 machines

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant exécuter ce fichier :

bash> docker-compose build bash> docker-compose up -d

Cette commande peut être un peu longue, c'est normal.
Vous pouvez vous connecter à la machine kali avec la commande :

bash> docker-compose exec -it kali /bin/bash
