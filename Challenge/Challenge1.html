Allez-vous réussir à trouver le contenu du fichier secret ?

Outils à installer

Installation de l'application Docker
Rendez vous sur votre invite de commandes
Il est necessaire de récuperer l'image Docker dans lequel vous devrez trouver le fichier secret

Pour cela :

- docker pull cybersecuriteach/challenge1:v0
- docker run -it cybersecuriteach/challenge1:v0
Puis on récupère l'image Docker representant la machine virtuelle kali.

Pour cela :

- docker pull cybersecuriteach/kali4fun:v0
- docker run -it cybersecuriteach/kali4fun:v0

Objectifs

Vous avez maintenant grâce aux 2 images recupérées une simulation d'attaques. La kali representera votre machine et l'image challenge1 representera la victime possédant le fichier secret.

L'objectif est donc de penetrer un serveur de la machine victime puis d'acceder au contenu du fichier secret.

A vous de jouer

Avec nos différents tutoriels et avec vos recherches personnelles vous devez être en mesure de trouver un message secret !
