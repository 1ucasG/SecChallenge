Challenge John the Ripper

Pour voir si vous avez bien compris le cours, je vous propose de vous entrainer sur un docker que j’ai créé spécialement pour cela.

Que devez-vous faire ?

Étape 1 : récupérer le docker proposé

Pour cela, il vous faut docker installé (il existe tout un tas de tutoriels qui vous expliquerons comment le télécharger) quand docker et docker-compose sont installés sur votre machine, vous pouvez commencer.
Il faut d'abord télécharger le fichier docker-compose.yml
Ensuite il vous suffit de taper la commande (en étant dans le bon répertoire où se trouve le fichier que vous venez de télécharger):

docker-compose up -d

Étape 2 : lancer la machine

Voilà ! Vous venez de télécharger les deux machines, il faut maintenant les lancer.
Pour lancer la machine de martin :

docker exec -it martin /bin/bash

Pour lancer la machine attaquante avec john installé :

docker exec -it john /bin/bash

Situation

Martin veut lire le contenu du fichier flag.txt présent sur sa machine, mais pour cela il doit être root sur sa machine. Malheureusement il ne connait pas son mot de passe administrateur car c’est son ami Jeremy qui a configuré sa machine ! Ce dernier ne se souvient plus du tout du mot de passe non plus :( Arriverez-vous à trouver le mot de passe administrateur de cette machine pour que martin puisse accéder au contenu du fichier flag.txt ?
