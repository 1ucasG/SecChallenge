Votre mission

Le but sera de trouver et faire en sorte de pouvoir lire secret permettant l'accès au flag. Petit indice: EternalBlue ;)

Les conditions de votre challenge:

Dans un premier temps, vous devez installer Docker. Pour ce faire, tapez dans votre terminal Linux: sudo apt install docker. Une fois que ce sera fait, tapez la ligne suivante: docker pull lightyx971/ctf_projet_e4:ss_challenge
Ces commandes permettent d'installer docker puis de télécharger votre environnement de recherche. Pour lancer le challenge: docker run -it ss_challenge.

Bonne recherche !
