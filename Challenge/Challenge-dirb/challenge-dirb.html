Trouvez le flag

Objectif

L'objectif de ce challenge est de trouver le fichier du site web http://Tourisme_Scranton/ contenant le flag.

Vous pourrez utiliser l'outil Dirb ainsi que wget.

Outils à installer

Il est nécessaire d'avoir installé l'application Docker et docker-compose
Ensuite, récupérez le fichier docker-compose.yml qui permet de lancer le serveur web du site et la machine attaquant depuis laquelle on cherchera le flag.

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant exécuter ce fichier :

bash> docker-compose build
bash> docker-compose up -d

Cette commande peut être un peu longue, c'est normal.

Vous pouvez ensuite vous connecter à la machine attaquant avec la commande : bash> docker exec -it ubuntu /bin/bash

Pour une meilleure interprétation vous avez accès depuis votre naviguateur à l'adresse http://localhost:5801 à un firefox sur lequelle vous pourrez accéder au site http://Tourisme_Scranton/
