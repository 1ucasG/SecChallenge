Trouvez le flag

Objectif

L'objectif de ce challenge est de trouver un fichier du site web http://bikeskryo/ contenant le flag.

Vous pourrez utiliser l'outil Dirb.

Prérequis

Il est nécessaire d'avoir installé l'application Docker et docker-compose

Il faut télécharger l'archive suivante : Challenge_dirb.rar

Puis dans le fichier décompressé de l'archive, exécuter les commandes suivantes en terminal:
chmod -R 777 ./db && chmod -R 777 ./wordpress
docker-compose -d --build

À vous de jouer

Pour lancer la machine de manipulation : docker exec -it ubuntu /bin/bash

Vous avez aussi accès au site bikeskryo depuis votre naviguateur à l'adresse http://localhost:8080

Avec notre tutoriel sur Dirb, vous devez être en mesure de retrouver le flag
