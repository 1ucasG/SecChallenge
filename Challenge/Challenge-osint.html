Allez-vous réussir à découvrir le contenu du fichier secret ?

Objectif

Grâce aux différentes informations que vous pourrez obtenir sur le réseau social de la cible, vous devrez déterminer le contenu du fichier secret.
L'objectif est donc de pratiquer une recherche d’information de type OSINT sur la cible puis d'accéder au contenu du fichier secret.

Outils à installer

L'application Docker est nécessaire
Il est également nécessaire de récuperer l'image de la kali.

Pour cela :

Télécharger l'image avec
bash> docker pull cybersecuriteachg2/osintchall:v1.0
Puis pour lancer la lancer
bash> docker run -it cybersecuriteachg2/osintchall:v1.0 /bin/bash

A vous de jouer

Avec nos différents tutoriels et avec le profil facebook de la victime vous devez être en mesure de trouver le mot de passe du dossier compressé.
