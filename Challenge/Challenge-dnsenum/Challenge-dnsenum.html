DNSEnum : Le Challenge


Configuration

Ce challenge met à disposition, par le biais de l'application Docker et docker-compose, trois machines connectées sur un même réseau local.
Il est nécessaire de récupérer le fichier docker-compose.yml qui permet de lancer les trois machines.
Téléchargez le docker-compose.yml
Les commandes qui permettent de lancer les 3 machines sont les suivantes :
bash> docker-compose up -d
Nous avons besoin de nous connecter à celle d'host 1 via la commande :
bash> docker exec -it dns_host1 /bin/bash


Votre objectif

Votre but est d’effectuer de l’énumération DNS sur le site de ‘challenge-foundation’ ! L’adresse ip de leur deuxième host nous intéresse particulièrement…


Renseignez le flag pour réussir le challenge !
