Challenge Stegseek

Manipulation de plusieurs fichiers steghide

Nous venons d'intercepter différents fichiers durant une conversation entre deux personnes lambda.
Nous pensons qu'ils communiquent entre eux de cette manière et que ces fichiers dissimulent des informations secrètes.
L'objectif est donc d'accéder au contenu secret final et de retrouver le message caché.

À vous de jouer !

Ce tutoriel met à disposition, par le biais de Docker et Docker-compose, une machine déjà configurée.
Récupérez le dockerfile dans le lien suivant : docker-compose.yml

Dans un bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez les commandes suivantes :
docker-compose build
docker-compose up -d
docker exec -ti challenge_052-syd-academy-stegseek /bin/bash
apt install ./stegseek_0.6-1.deb

Avec nos différents tutoriels et vos recherches personnelles, vous devez être en mesure de récupérer un flag !

NB : Quelques indices si vous êtes perdus

Pour valider ce tutoriel, il vous suffira de rentrer le message secret du dernier fichier steghide.
