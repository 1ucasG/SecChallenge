Interception de mot de passe

Objectif

L'objectif de ce challenge est de mettre en pratique le cours sur le man in the middle
Vous prendrez le controle de Eve qui doit intercepter la requête http que Alice effectue pour aller sur le site de Bob

Outils à installer

Installation de l'application Docker et docker-compose
Rendez vous sur votre invite de commandes
Il est nécessaire de récuperer le fichier docker-compose.yml qui permet de lancer les 3 machines

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant éxecuter ce fichier:
bash> docker-compose build
bash> docker-compose up -d
Cette commande peut être un peu longue, c'est normal.
Les trois machines sont lancées et vous pouvez vous connecter à la machine de Eve avec la commande:
bash> docker exec -it mitm_eve /bin/bash

A vous de jouer

Avec nos différents tutoriels et avec vos recherches personnelles vous devez être en mesure de trouver le mot de passe de Alice
