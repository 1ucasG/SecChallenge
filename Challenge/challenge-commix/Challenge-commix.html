Challenge Commix

Objectif

L'objectif de ce challenge est de mettre en pratique le cours et le tuto sur l'utilisation de Commix.

Outils à installer

Installation de l'application Docker et docker-compose
Télécharger le fichier docker-compose.yml
Exécutez les commandes suivantes :
bash> docker-compose up --build
Cette commande peut être un peu longue, c'est normal.
Pour se connecter à la machine :
bash> docker-compose exec -ti commix /bin/sh
La page web est accessible à l'adresse suivante :
http://192.1618.60.3:5000

À vous de jouer !
