Challenge CeWL

Objectif

L'objectif de ce challenge est de mettre en pratique le cours et le tuto sur l'utilisation de CeWL

Outils à installer

Installation de l'application Docker et docker-compose
Il est nécessaire de récupérer le fichier docker-compose.yml

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant exécuter ce fichier :
bash> docker-compose build
bash> docker-compose up -d
Cette commande peut être un peu longue, c'est normal.
Pour se connecter à la machine :
bash> docker-compose exec -it cewl /bin/bash
Pour voir le site web, tapez dans un navigateur :
http://localhost:4444
Le site web à attaquer depuis la machine est le 192.168.1.3:4444/secret.php

À vous de jouer !
