Injection SQL

Objectif

L'objectif de ce challenge est de mettre en pratique le cours et le tuto sur les injections SQL
Vous prendrez le contrôle d'une machine kali

Outils à installer

Installation de l'application Docker et docker-compose
Rendez-vous sur votre invite de commandes
Il est nécessaire de récupérer le fichier docker-compose.yml qui permet de lancer les 3 machines

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant exécuter ce fichier :
bash> docker-compose build
bash> docker-compose up -d
Cette commande peut être un peu longue, c'est normal.
Pour se connecter à la machine :
bash> docker-compose exec -it mitm_eve /bin/bash
Pour voir le site web, tapez dans un navigateur :
http://localhost:3001
