Trouver le message caché sous l'image

Outils à installer

Installation de l'application Docker
Il est necéssaire de récupérer le fichier Steganographie.zip que vous pourrez récupérer à l'adresse suivante : https://github.com/secyourdev/SecChallenge/tree/Elies
Rendez-vous sur votre invite de commandes
IMPORTANT Avec la commande cd positionnez vous à l'endroit dans lequel vous avez dézippé le fichier Steganographie.zip.
(Par exemple cd C:\Users\asus\Documents\Steganographie)
Pour lancer :
- docker build -t kali:v0 -f .\Dockerfile.txt .
- docker run -it kali0
Maintenant déplacer vous dans les fichiers pour vous rendre dans le répertoire home/msfadmin.
Tapez la commande suivante : steghide embed -ef crack.txt -cf Cybersecurity.jpg -sf cybersecret -p mdp

Objectifs

Vous disposez maintenant d'une image se nommant cybersecret. L'objectif est de retrouver le message caché sous l'image.
Une fois que vous pensez avoir trouvé, verifiez votre réponse !

À vous de jouer

Avec nos différents tutoriels et avec vos recherches personnelles, vous devez être en mesure de
trouver un message secret !
