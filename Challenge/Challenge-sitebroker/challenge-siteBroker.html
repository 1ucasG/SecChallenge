Challenge: Scanner un site



Pull it, search it



Dans ce challenge, vous allez vous familiariser avec l'outil SiteBroker. Vous utiliserez une machine docker. Pour cela, installez docker avec la commande apt install docker, puis télécharger la machine avec: docker pull cybersecuriteachg2/sitebroker Derniez réglage: lncez la machine avec un docker run -it challenge_wget. Ce challenge est simple: un CTF (capture the flag), dans lequel vous devrez trouver le message caché sur un site hébergé en local.
Pour cela, vous devrez trouver comment utiliser correctement l'outil, ainsi que les lignes de commandes de base de linux. Indice: Sur tout site, l'admin possède une page html dédiée ;) Bon courage!
