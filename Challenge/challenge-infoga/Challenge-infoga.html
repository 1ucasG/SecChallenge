Challenge Infoga

Objectif

L'objectif de ce challenge est de mettre en pratique le cours sur l'utilisation de Infoga

Outils à installer

Installation de l'application Docker et docker-compose
Il est nécessaire de récupérer le fichier docker-compose.yml

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant exécuter ce fichier :
bash> docker-compose up --build
Cette commande peut être un peu longue, c'est normal.
Pour se connecter à la machine :
bash> docker exec -it infoga /bin/bash
Pour voir le site web du challenge OSINT, tapez dans un navigateur :
http://localhost:80

À vous de jouer !
