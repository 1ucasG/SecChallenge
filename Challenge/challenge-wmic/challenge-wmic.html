Challenge John the Ripper

Pour voir si vous avez bien compris le cours, nous allons simplement essayer de vous faire retrouver une commande en utilisant la commande help de wmic.

Que devez-vous faire ?

Étape 1 : Ouvrir le logiciel invité de commandes

Pour faire celà il vous suffit comme expliqué dans le cours d'aller dans la barre de recherche Windows si vous êtes sur une machine Windows et de trouver une des deux applications. Sinon vous devez télécharger une machine Windows virtuelle.
Il faut d'abord se procurer une machine windows virtuel si vous posséder une machine Linux : machine-virt.yml

Étape 2 : Création et supression du compte Windows

Voilà ! Vous venez de lancer le logiciel invité de commandes.
Pour Ajouter un compte Windows :
net user nomutilisateur motdepasse /add
Pour voir les utilisateurs windows:
wmic useraccount list full
Pour Enlever un compte Windows :
net user nomutilisateur motdepasse /delete

Situation

Votre compte Windows a eu dernièrement un léger soucis et vous souhaitez ajouter un utilisateur Windows pour pouvoir le remplacer mais sans passer par le menu de gestion des comptes Windows. Vous allez donc devoir passer par la fonction Wmic de Windows. La réponse que vous devrez donner sera sous forme de haché sha1 (utilisez cette adresse) :sha-1.yml
