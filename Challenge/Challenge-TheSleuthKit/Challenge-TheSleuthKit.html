The Sleuth Kit : Le Challenge

La mission :

Bienvenue sur la page du challenge The Sleuth Kit ! Votre mission est la suivante :

Vous possédez une image disque d'une clé usb d'un inconnu, et vous devez retrouver son identité ! En particulier, son adresse mail pour pouvoir le recontacter.

Prérequis :

Décompressez le zip contenant l'image : imageChallenge.zip
Vous pourrez ensuite utiliser la machine via : docker pull tsk-challenge

Bon courage !
