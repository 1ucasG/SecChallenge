Challenge RainbowCrack

Que devez-vous faire ?

Étape 1 : récupérer le docker proposé

Pour cela, il vous faut docker d'installé (il existe tout un tas de tutoriels qui vous expliquerons comment le télécharger) quand docker est installé sur votre machine, vous pouvez commencer.
Il vous suffit de taper la commande suivante afin de récupérer l'image nécessaire au challenge :
docker pull tomdcn/rainbowcrack

Étape 2 : lancer la machine

Voilà ! Vous venez de télécharger la machine, il faut maintenant la lancer.
Pour lancer la machine, tapez la commande suivante : docker run -it tomdcn/rainbowcrack

Situation

Vous avez désormais toutes les cartes entre vos mains pour déchiffrer le code d'accès que vous avez récupéré sous le fichier mdp.txt.
