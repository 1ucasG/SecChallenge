Allez vous réussir à exploiter le token JWT et voler la session administrateur ?

Objectif

L'objectif de ce challenge est d'exploiter un token JWT. Cela va permettre d'usurper la session administrateur.

Outils à installer

Installation de l'application Docker et docker-compose
Rendez-vous sur votre invite de commandes
Il est nécessaire de récupérer le fichier docker-compose.yml qui permet de lancer le client et le serveur web.

Pour cela :

Télécharger le docker-compose.yml
Il faut maintenant lancer le container :
bash>docker-compose up --build
Cette commande peut être un peu longue, c'est normal.
Ensuite, vous avez accès au site web : http://localhost:3000
