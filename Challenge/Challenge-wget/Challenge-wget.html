Hands-On: WGET



Pull it, search it



Dans ce challenge, vous allez vous familiariser avec l'outil wget. Vous utiliserez une machine docker. Pour cela, installez docker avec la commande apt install docker, puis télécharger la machine avec: docker pull cybersecuriteachg2/challenge_wget Derniez réglage: lncez la machine avec un docker run -it challenge_wget, puis quand celle-ci est lancée, tapez service apache2 start Ce challenge est simple: un CTF (capture the flag), dans lequel vous devrez trouver le message caché sur un site hébergé en local.
Pour cela, vous devrez trouver comment utiliser correctement l'outil, ainsi que les lignes de commandes de base de linux. Bon courage!
