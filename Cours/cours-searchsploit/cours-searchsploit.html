Searchsploit

Description

Searchsploit est un outil utilisable en ligne commande, qui permet de faire des recherches d'exploits, en étroite relation avec le site et la base de données Exploit-DB. Il permet d'"obtenir" un exploit viable en fonction d'une version d'un logiciel. Permet aussi de prendre l'habitude d'utiliser les lignes de commandes ;)

Note: Ces exploits sont à evidemment utiliser dans un cadre sécuriser et légal.

principe/b>

Permet une recherche rapide d'un exploit en fonctions des versions et possède aussi d'autres fonctionnalités tels que la copie de la base de données Exploits DB, mais possède aussi d'autres utilisations.
