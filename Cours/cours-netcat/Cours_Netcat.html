Netcat

1. Objectif

L'objectif de ce cours et de vous présenter l'outil Netcat ainsi que certaines de ces fonctionnalités.

Netcat est un outil de réseaux qui est principalement utilisé pour permettre la communication entre plusieurs machines en utilisant des connexions TCP ou UDP.
On peut utiliser Netcat pour notamment scanner des ports, transférer des fichiers ou encore faire du reverse shell. Vous pouvez afficher la page de manuel de Netcat en entrant netcat -h ou nc -h dans un terminal.

2. Configuration

L'installation de Netcat sur un système Linux est très simple, il suffit d'ouvrir un terminal et d'entrer la commande apt install netcat-traditional.
Il est possible que la commande "sudo" soit nécessaire, ce qui donnera sudo apt install netcat-traditional.

3. Les Utilisations

I. Transfert de fichier

Il est possible d’envoyer des fichiers entre différentes machines en utilisant Netcat.
Le transfert de fichier avec Netcat se fait en utilisant les ports d’une machine pour ouvrir un canal de communication entre les machines. La méthode d’envoi et la suivante:

Sur la machine réceptrice, on entre la commande suivante dans un terminal:
netcat -l -p [port_number] > [file_name]
Il faut entrer la valeur du port que l'on a choisi d'utiliser pour la communication dans le champs [port_number].
La partie [file_name] correspond au fichier qui va stocker les informations qui circuleront à travers le port choisi.

Nous allons ensuite nous rendre sur la machine émettrice et saisir la commande suivante:
netcat -w [Temps] [receiver_ip] [port_number] < [FileToSend]
La valeur [Temps] correspond au nombre de secondes avant que la connexion soit automatiquement coupée, receiver_ip est l’adresse IP de la machine sur laquelle on envoie le fichier.
La valeur du [port_number] doit être la même que celle choisie sur la machine réceptrice.
Le champs [FileToSend] correspond au fichier que l’on souhaite envoyer.

II. Scan de port

Netcat, à l’instar de Nmap, offre la possibilité de scanner les ports d’une machine.
Pour ne scanner qu'un seul port, on peut utiliser la commande suivante: netcat -z -v [ip_adress] [port_number]
Cette commande va scanner le port numéro [port_number] et indiquer s’il est ouvert ou non.
Mais on peut aussi scanner plusieurs ports à la fois en utilisant la commande suivante: netcat -z -v [ip_adress] [range]
Le principe est exactement le même qu'avec le scan d'un seul port, sauf que cette fois-ci, on entre un interval de ports à scanner. Par exemple, la commande netcat -z -v [ip_adress] 1-50 va scanner tous les ports numérotés de 1 à 50.
